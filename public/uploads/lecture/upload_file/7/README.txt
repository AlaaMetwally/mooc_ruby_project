Bash scripting (DBMS):
-----------------------
1) clone/download project in $HOME
2) open terminal in $HOME
3) Run bash start.sh , you can run this command in any path/directory

Our DBMS based on Command lines

syntax of our DBMS:
--------------------
1) create database dbname
2) rename table oldname to newname
3) desc tbname
4) create table tablename
5) sort table tablename
6) drop table tablename
7) use databasename
8) show databases
9) show tables
10) alter table tablename drop column colname
11) alter table tablename add colname datatype
12) alter table tablename alter column colname datatype
13) insert into tablename
14) select from tablename 
15) edit record from tablename 
15) edit records from tablename 
16) delete from tablename
